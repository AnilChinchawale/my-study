# How to install Go compiler (on Ubuntu)
1. install GO compiler
<pre>
$ sudo apt-get install gccgo-go
</pre>

#How to run code without compiling
<pre>
$ go run hello.go
</pre>

# How to build code
1. build `hello.go`
<pre>
$ go build ./hello.go
</pre>

# How to run code
1. run `hello`
<pre>
$ ./hello
</pre>
